Reccomendations¶

  • Producers should focus their efforts on producing low budget horror movies given the superior return profile.
  • Producers shouldn't focus on vote averages, since the data clearly shows this doesn't affect the return profiles.
  • Producers should focus on making lower budget films, since higher budget films aren't warranted from a returns perspective.
